# Metrotherapy Brand Pack

Состав:
- `botfather_config_ru.txt` — тексты для BotFather.
- `start_message_ru.txt` — текст приветствия `/start`.
- `telegram_buttons_ru.txt` — рекомендуемые кнопки главного меню.
- `index.html` — минимальный лендинг.
- `metrotherapy_brand_pack.zip` — архив с бренд-паком.

Важно:
- `artifacts/` остаётся локальной временной папкой и не коммитится.
- Каноничная версия бренд-пака в GitHub лежит здесь: `docs/brand/`.
